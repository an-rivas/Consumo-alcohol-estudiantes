# Consumo-alcohol-estudiantes

Proyecto Final de la materia Minería de Datos de la Licenciatura en Tecnologías para la Información en Ciencias de la ENES Morelia en el semestre 2020-1.

Este proyecto busca crear un modelo que pueda categorizar estudiantes de acuerdo a su consumo de alcohol semanal. 

As a final project for the data mining course, the aim is to create a model that can categorize students according to their weekly alcohol consumption.
